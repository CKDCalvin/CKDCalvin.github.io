The artifact selected for this enhancement is the Pet Boarding System originally developed in IT 145. The original project was a Java-based object-oriented program designed to manage pet boarding information such as pet type, capacity, and reservations. While functional, the original version relied on simple data handling and did not include advanced algorithms for managing reservations or optimizing system behavior.

---
For this enhancement, the artifact was redesigned and extended into a full-stack MERN (MongoDB, Express, React, Node.js) application. The updated system implements algorithmic logic and improved data structure usage to manage reservations, assign suites, and provide efficient search and sorting capabilities for administrative use. This artifact was selected because it provides a strong opportunity to demonstrate practical applications of algorithms and data structures within a real-world system. The original implementation lacked dynamic decision-making and efficient data handling, making it an ideal candidate for enhancement. 

---
The enhanced system showcases several key algorithmic components. First, a suite assignment algorithm was implemented to allocate available boarding spaces based on pet type. This algorithm iterates through available suites and assigns the first valid space, ensuring that capacity constraints are respected and overbooking is prevented. Additionally, search and sorting algorithms were introduced to improve administrative efficiency. The system supports searching reservations by both reservation ID and pet name using string matching techniques. Sorting functionality allows reservations to be organized by pet name, pet type, or check-out date, improving usability and data accessibility. The suite assignment algorithm operates with linear time complexity O(n), while sorting operations follow O(n log n), demonstrating consideration of efficiency and scalability. These enhancements significantly improve the system by transforming it from static data handling into a dynamic, algorithm-driven application that reflects real-world operational needs.
