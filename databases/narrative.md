---
The artifact selected for this enhancement is the Pet Boarding System originally developed in IT 145. The original project was a Java-based, object-oriented program designed to manage pet boarding information, including pet type, capacity, and reservations. While the system demonstrated foundational object-oriented programming concepts, it lacked a persistent data storage layer. Reservations were not permanently stored, and users were unable to update or delete existing reservations once they were created.

---
This artifact was selected for inclusion in my ePortfolio because it provided a strong foundation to demonstrate growth in database design and full-stack development. By enhancing the project using a MERN-based architecture, I was able to transform it from a static, in-memory application into a dynamic system backed by MongoDB. Key components of this enhancement include the implementation of full CRUD (Create, Read, Update, Delete) functionality, structured data models for reservations and suites, and integration between the frontend, backend, and database layers. The addition of updating and deleting capabilities, along with validation and suite reassignment logic, showcases my ability to design and implement real-world database-driven applications. These improvements significantly increased the functionality, usability, and scalability of the original system.

---
Through this enhancement, I met the course outcomes related to designing and evaluating computing solutions using appropriate tools and techniques, as well as implementing database-backed systems that deliver value. I demonstrated the ability to use modern technologies such as MongoDB, Mongoose, and RESTful APIs to manage persistent data. Additionally, I strengthened my understanding of integrating backend services with frontend applications to create a complete user experience. While the primary focus of this enhancement was database implementation, it also supported outcomes related to software engineering practices and system design.

---
The process of enhancing this artifact provided valuable learning experiences. One of the most significant lessons was understanding how to manage data persistence and maintain consistency between related data, such as ensuring that a suite becomes available again when a reservation is deleted. I also gained experience designing API routes for CRUD operations and handling asynchronous data flow between the client and server. A key challenge I encountered was debugging issues related to data formatting and state updates, particularly with date handling and ensuring that updates were correctly reflected in the user interface. Overcoming these challenges improved my problem-solving skills and reinforced the importance of careful testing and validation. Overall, this enhancement allowed me to transform a basic object-oriented program into a fully functional, database-driven application that reflects industry practices.
