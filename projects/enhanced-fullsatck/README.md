# Posh Paws Boarding — Full-Stack Application

## Overview

The Posh Paws Boarding application is a full-stack web system developed as an enhancement of the original IT 145 Pet Boarding System. This version transforms a simple object-oriented Java program into a dynamic, database-driven application using the MERN stack (MongoDB, Express, React, Node.js).

Customers can register, reserve pet boarding spaces, and retrieve reservation details, while also enabling administrative management of reservations through search, sorting, and CRUD functionality.

## Features

### Customer Features

* User registration and login
* Reservation creation for dog or cat boarding
* Automatic suite assignment based on availability
* Reservation lookup using the reservation ID

### Admin Features

* View all reservations
* Search by reservation ID, pet name, or pet type
* Sort reservations by pet name, type, or check-out date
* Update existing reservations
* Delete reservations
