# IT 145 Pet Boarding System (Original Artifact)

## Overview

The IT 145 Pet Boarding System is a Java-based object-oriented application developed to manage pet boarding operations. The system allows users to create and manage pet reservations, track capacity, and simulate basic boarding functionality.

This project served as the foundational artifact for the capstone enhancement into a full-stack application.

---

## Source Code

🔗 https://github.com/CKDCalvin/IT-145-PetBoardingSystem

---

## Features

* Create pet boarding reservations
* Track pet type (dog or cat)
* Manage boarding capacity
* Simulate check-in functionality

---

## Technologies Used

* Java
* Object-Oriented Programming (OOP)

---

## Limitations

* No persistent data storage (in-memory only)
* No graphical user interface
* No ability to update or delete reservations
* Limited scalability

---

## Role in Capstone

This project was selected as the foundation for enhancement due to its clear structure and potential for expansion.

The capstone enhancement transformed this system into a full-stack application by introducing:

* A modern user interface
* Database integration
* Advanced algorithms
* Real-world application architecture
