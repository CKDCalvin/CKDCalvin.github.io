# Enhancement One Narrative: Software Design and Engineering
---
The artifact selected for this enhancement is the IT 145 Pet Boarding System, which was originally developed as an object-oriented Java application earlier in the program. For this milestone, the project was redesigned and enhanced into a full-stack web application using React for the frontend and Node.js with Express for the backend. The enhanced system now includes: a user interface with multiple pages, routing functionality, and a working check-in system that allows users to register and log in. This transformation shifted the artifact from a console-based application to an interactive, web-based system that better reflects modern software development practices.
---
I selected this artifact for inclusion in my ePortfolio because it demonstrates a wide range of software engineering and design skills. Specifically, it showcases my ability to refactor an existing application into a new architecture, implement a client-server model, and design a user-friendly interface. Key components of the artifact that highlight my skills include the integration of React components, the use of routing for navigation, and the implementation of API communication between the frontend and backend using Axios. Additionally, the system now supports user registration and login functionality, which required designing form handling, state management, and backend route handling. The artifact was enhanced by transitioning from a static, object-oriented program into a dynamic, full-stack application, incorporating modular design, reusable components, and improved user interaction.
---
Through this enhancement, I made strong progress toward several course outcomes identified in Module One. In particular, I demonstrated the ability to design and implement computing solutions using modern tools and frameworks, aligning with software engineering and design practices. I also improved my ability to create technically sound and well-structured applications that are adaptable to real-world use cases. While this enhancement primarily focused on software design and engineering, it also began to support outcomes related to database interaction through user authentication and data validation. Moving forward, I plan to expand this artifact to further address outcomes related to algorithms and data structures by implementing efficient search and sorting features, as well as deeper database integration.
---
The process of enhancing and modifying this artifact provided valuable learning experiences and presented several challenges. One of the most significant challenges was transitioning from a Java-based object-oriented program to a full-stack JavaScript application, which required learning new frameworks and understanding how frontend and backend systems communicate. Another challenge involved managing CSS styling across multiple pages. Initially, the global styles were unintentionally applied to elements such as headings, causing inconsistencies between pages. To resolve this, I refactored the styling approach by converting CSS files into module-based stylesheets (filename.module.css) and applying scoped styles using imported class names. This ensured that styles were isolated to their respective components, enhancing maintainability. Overall, this enhancement strengthened my understanding of component-based architecture, state management, routing, and full-stack development, while also reinforcing the importance of debugging, organization, and scalable design practices.
